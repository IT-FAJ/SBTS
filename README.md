# 🚌 School Bus Tracking System (SBTS)

🚀 A smart transportation management system for tracking school buses in real-time and improving student safety.

---

## 📌 Overview

SBTS is a web-based system designed to:

- 📍 Track school buses live using GPS  
- 🎫 Automate student attendance  
- 🔔 Send real-time notifications  
- 🔐 Secure access with role-based authentication  

---

## 🧰 Tech Stack

### 🔙 Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- bcryptjs
- dotenv

---

## 👥 User Roles

- 👨‍💼 Admin  
- 🧑‍✈️ Driver  
- 👨‍👩‍👧 Parent  

Each role has controlled access through secure authentication and authorization.

---

## 🔐 Features Implemented

- 🛡 Secure login system
- 🎟 JWT-based authentication
- 🔒 Protected API routes
- 🚦 Role-Based Access Control (RBAC)
- ⛔ Suspended account blocking
